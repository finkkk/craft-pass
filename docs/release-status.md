# 当前完成状态

Craft Pass 当前已经具备可部署的核心版本：

- 玩家资料填写、服规阅读、签署与动态题库答题
- 后端独立校验、判分、协议版本检查与重复申请限制
- 管理员登录、会话鉴权、申请分类、详情、修改和删除
- 审核拒绝、RCON 白名单执行、失败记录与重试
- 站点名称、Logo、RCON、服规、签署声明、题库和合格线后台配置
- 申请统计、状态分布、答题/审核/RCON 成功率与趋势
- 首次网页部署向导、单端口生产构建和一键部署命令
- 管理员操作日志、敏感配置加密、限流和统一错误响应

## 发布前仍需外部环境确认

- 使用真实 Minecraft 服务端验证 RCON 地址、密码、命令返回文本和网络访问策略。
- 公网部署时配置域名、HTTPS、反向代理或进程守护，并备份整个 `backend/data`。

## 当前未包含的可选扩展

- 多管理员账号、角色和权限管理页面
- 玩家使用 QQ 号与 Minecraft ID 自助查询历史申请状态
- Docker Compose 等特定平台部署模板

以上项目不阻塞当前核心申请、审核和白名单流程。
