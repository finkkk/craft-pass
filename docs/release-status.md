# 当前完成状态

Craft Pass 当前已经具备可部署的核心版本：

- 玩家资料填写、服规阅读、签署、随机抽题与防篡改判分
- 通过答题后 QQ/Minecraft ID 唯一绑定、答题失败可重考、冲突提示和组合进度查询
- 管理员登录、会话鉴权、申请分类、详情、修改和删除
- 审核拒绝、RCON 白名单执行、输出记录、失败记录与重试
- 多选批量批准、拒绝和 RCON 重试
- RCON 连接状态检测与后台自定义命令控制台
- 站点名称、Logo、玩家端界面文案、RCON、服规、签署声明、题库和合格线分区后台配置
- 申请入口开关、通过答题后 QQ/Minecraft ID 查重和同 IP 提交频率限制
- 自定义 RCON 命令开关与危险命令黑名单
- 恢复出厂设置并重新进入首次部署流程
- 申请统计、状态分布、答题/审核/RCON 成功率与趋势
- 首次网页部署向导、单端口生产构建和一键部署命令
- 后台站内导航、前后台代码拆分、配置缓存和静态资源长期缓存
- GitHub Releases 定时检查和后台版本更新提醒
- 管理员操作日志、敏感配置加密、限流和统一错误响应

## 发布前仍需外部环境确认

- 使用真实 Minecraft 服务端验证 RCON 地址、密码、命令返回文本和网络访问策略。
- 公网部署时配置域名、HTTPS、反向代理或进程守护，并备份整个 `backend/data`。

## 当前未包含的可选扩展

- 多管理员账号、角色和权限管理页面
- Docker Compose 等特定平台部署模板

以上项目不阻塞当前核心申请、审核和白名单流程。
